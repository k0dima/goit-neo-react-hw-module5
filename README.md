# goit-neo-react-hw-module5

Movie search application built with React, Vite, React Router, axios and CSS Modules.

## Setup

Install dependencies:

```bash
npm install
```

Create `.env` in the project root and add your TMDB API Read Access Token:

```env
VITE_TMDB_TOKEN=your_tmdb_api_read_access_token
```

Run the project:

```bash
npm run dev
```

Build the project:

```bash
npm run build
```
